# energy_platform_spider
## 机电事业部能源数据爬取软件                    

该软件能爬取[http://energy.easynx.com/Energy/](http://energy.easynx.com/Energy/) 后台管理界面中，统计分析选项下所有数据，并定期（一般为15min）上传到oracle数据库。                     
本软件遵守GPLv3协议。
